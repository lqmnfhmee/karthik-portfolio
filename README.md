# Karthik Portfolio Website

A personal portfolio website created for **Karthik Raj A/L Kannan**, a full-time student, to showcase his background, academic projects, and contact information. This website is developed using **pure HTML, CSS, and JavaScript** as part of the **Web Application Development (TMT3693)** assignment.

Live Website:  
- https://lqmnfhmee.github.io/karthik-portfolio/

---

## Project Overview

This portfolio website is designed to provide a clean and responsive interface where visitors can learn more about Karthik, explore his projects, and reach out through the contact section. The website emphasizes simplicity, usability, and pastel-themed aesthetics, with the option for a fully functional **dark mode**.

---

## Features

### **Frontend Design**
- Fully responsive for mobile, tablet, and desktop
- Pastel-themed color palette  
- Smooth UI transitions and hover effects  
- Multi-page structure (Home, About, Projects, Contact)

### **Dark Mode**
- Toggle between light and dark themes  
- Dark mode preference is saved automatically using `localStorage`  
- Applied persistently across all pages  

### **Typing Animation**
- Dynamic typing effect on the Home page  
- Cycles through phrases describing the client

### **Scroll Reveal Animations**
- Elements smoothly fade/slide into view as the user scrolls  
- Enhances visual engagement and professionalism

### **Scroll-to-Top Button**
- Appears when scrolling down  
- Smoothly scrolls the page back to the top  

### **Contact Form with Validation**
- Real-time client-side validation  
- Checks name, email format, and message length  
- Displays success/error messages without reloading the page

## Deployment

This website is deployed using **GitHub Pages**.  
To deploy your own version:

1. Upload your project files to a public GitHub repository.
2. Go to **Settings → Pages**.
3. Under *Branch*, select `main` and root `/`.
4. Save and wait 1–3 minutes for deployment.
